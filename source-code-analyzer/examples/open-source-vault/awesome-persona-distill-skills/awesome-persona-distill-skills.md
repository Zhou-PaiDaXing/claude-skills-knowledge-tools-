## 项目简介

这是一个围绕人物、关系、纪念性场景与方法论视角的 [Agent Skills](https://agentskills.io) 收录列表。项目聚焦于"人格蒸馏"(Persona Distillation)——从对话、作品、资料或数字痕迹中提炼表达风格、决策框架与交互方式。

**重要声明**：这里的"人格蒸馏"不默认等同于对真实个体的完整还原，而是提取可复用的表达模式和思维框架。

## 关键特点

### 1. 收录标准
- 从人物、关系、对话、作品或数字痕迹中提炼
- 将人物视角、关系经验或方法论整理成可复用技能
- 具有明确主题边界

### 2. 提交流程
```
提交 Issue 表单 → 维护者审核 → 添加 approved 标签 → 工作流自动生成 PR
```

```markdown

```

### 如何使用

**作为用户**：
1. 浏览列表找到感兴趣的 skill
2. 访问对应 GitHub 仓库
3. 按照该 skill 的说明进行安装和使用

**作为贡献者**：
1. 阅读 [CONTRIBUTING.md](https://github.com/xixu-me/awesome-persona-distill-skills/blob/main/CONTRIBUTING.md)
2. 提交 Issue 表单申请新增收录
3. 等待审核和自动 PR 生成

**作为开发者**：
1. 参考现有 skill 的结构和设计模式
2. 开发自己的 persona skill
3. 提交到本列表获得曝光

## 相关概念

### Agent Skills
- 网站: https://agentskills.io
- 定义: 可复用的 AI 智能体能力单元
- 通常以 Markdown 格式描述，包含角色定义、能力说明、使用示例

### Persona Distillation (人格蒸馏)
- 从各种来源提取表达风格、决策框架、交互模式
- 不同于简单的角色扮演，强调提炼可复用的方法论
- 来源包括：对话记录、作品、社交媒体、数字痕迹等
