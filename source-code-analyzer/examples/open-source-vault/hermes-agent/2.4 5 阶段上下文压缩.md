# 2.4 5 阶段上下文压缩

> 📚 来源:[[hermes-agent-learning-plan#2.4 5 阶段上下文压缩|学习计划 · 阶段二 · 2.4]]
> 🧭 相关:[[2.3 Memory 系统]] · [[2.5 主循环 run_conversation]]

---

## 核心思路

5 阶段渐进式压缩,前 3 阶段**不调用 LLM**(免费且无幻觉风险),第 4 阶段才使用 LLM 摘要,第 5 阶段做完整性修复。

| Phase | 动作 | 是否调用 LLM |
|-------|------|--------------|
| Phase 1 | 裁剪工具结果(超长输出截断) | 否 |
| Phase 2 | 头部保护(保留系统/任务起点) | 否 |
| Phase 3 | 尾部保护(保留最近 N 轮对话) | 否 |
| Phase 4 | LLM 摘要中间 messages | 是 |
| Phase 5 | 修复孤立 tool_use / tool_result 配对 | 否 |
