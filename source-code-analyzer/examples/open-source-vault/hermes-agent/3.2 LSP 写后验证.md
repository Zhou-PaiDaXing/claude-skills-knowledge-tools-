## 三层防护

| 层 | 时机 | 反馈 |
|----|------|------|
| **in-process syntax check** | 写文件后立即 | 语法错误(免费、零延迟) |
| **LSP delta diagnostics** | 写文件后通过 LSP 询问 | **只返回新增错**(before-write baseline diff) |
| **per-turn mutation footer** | 每轮结束 | 注入 user msg:"写了 foo.py +15/-3" |

---

## 关键设计:before-write baseline diff

写文件前先记录 baseline 诊断,写完后再次诊断,**只把"新增的错"**返回给 Agent。
- ✅ 避免噪声(已有错误不重复提醒)
- ✅ 让 Agent 专注于"我刚才搞坏了什么"
- ✅ 修复诱因:Agent 不会被未修过的代码报错刷屏

---

## 代码入口

- `agent/lsp/manager.py` — LSP client 管理(多 language server 并发)
- `tools/file_operations.py` — 写文件钩子,调用 LSP 做 baseline / after diff
