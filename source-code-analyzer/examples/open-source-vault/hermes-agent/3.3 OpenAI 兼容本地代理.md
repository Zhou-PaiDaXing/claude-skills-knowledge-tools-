# 3.3 OpenAI 兼容本地代理 🆕 v0.14

> 📚 来源:[[hermes-agent-learning-plan#3.3 OpenAI 兼容本地代理 🆕 v0.14|学习计划 · 阶段三 · 3.3]]
> 🧭 相关:[[2.2 Provider-Transport 双层抽象]]

---

## 核心思路

**Credential-attaching forwarder**——把 Claude Pro / SuperGrok 的**订阅级 OAuth**反向暴露成 OpenAI 兼容接口,让任何 OpenAI 客户端(Codex / Aider / Cline 等)直接复用订阅。

```
Codex/Aider/Cline ──OpenAI API──▶  hermes proxy(本地)  ──OAuth + 转换──▶  Claude Pro / SuperGrok
```

---

## 价值

- 复用已有订阅(不必为 IDE 工具单独买 API key)
- OpenAI 客户端零改动接入
- 走本地代理,避免 token 泄露给第三方

---

## 代码入口

- `hermes_cli/proxy/server.py`(**308 行**)
