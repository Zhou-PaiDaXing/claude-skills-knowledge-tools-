> 📚 来源:[[hermes-agent-learning-plan#3.1 三层闭环 ⭐⭐⭐ (v0.12/v0.13)|学习计划 · 阶段三 · 3.1]]

---

## 三层叠加结构

| 层 | 时间尺度 | 触发 | 作用 |
|----|----------|------|------|
| **Background Review** | 每轮(turn) | run_conversation 每完成一轮后 fork | 实时评分 + 错误信号 |
| **Curator** | 7 天周期 | 后台 cron | 7 天周期归档清理:ACTIVE → STALE(30d) → ARCHIVED(90d) |
| **Multi-agent Kanban** | 多 Agent 协作生命周期 | 跨 Agent 协调 | durable 看板 + Hallucination Gate |

---

## 3.1.1 Background Review

- **代码**:`agent/background_review.py`(**570 行**)
- **核心**:每完成一轮,fork 一个评估子 Agent,对刚写的代码/工具结果打分。不阻塞主流程
- **价值**:把"事后 review"变成"事中 review",写完一轮立即拿到改进信号

## 3.1.2 Curator

- **代码**:`agent/curator.py`(**1781 行**)
- **核心**:7 天周期状态机,自动把陈旧的 memory / skill 标记归档(ACTIVE → STALE → ARCHIVED)
- **价值**:防止 memory 系统无限膨胀,保持上下文精简

## 3.1.3 Multi-agent Kanban

- **代码**:`plugins/kanban/`
- **核心**:durable 看板 + Hallucination Gate(防止 Agent 幻觉式认领任务)
- **价值**:多 Agent 协作时的状态共享与去重
