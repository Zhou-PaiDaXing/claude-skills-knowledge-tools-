# A2: RAG系统分析

## 分析框架

### 1. RAG架构
索引流程：文档→解析→分块→向量化→存储
查询流程：查询→检索→重排序→生成

### 2. 文档处理
解析方式、分块策略（固定/句子/段落/语义）、块大小和重叠参数。

### 3. Embedding策略
模型选择、混合检索（稠密+稀疏）、缓存策略。

### 4. 向量存储
存储选型(Pinecone/Milvus/Qdrant/Chroma)、索引类型(HNSW/IVF)。

### 5. 检索与重排序
基础检索方式、查询扩展、重排序模型、多样性控制(MMR)。

### 6. 生成策略
Prompt模板、上下文组装、引用追溯、幻觉控制。

### 7. 可借鉴
前3个RAG优化技巧。
