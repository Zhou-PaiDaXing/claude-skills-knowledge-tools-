# A1: Agent架构分析

## 分析框架

### 1. Agent核心架构
感知模块→决策模块→执行模块，记忆模块、规划模块。
绘制架构图。

### 2. 决策机制
推理模式：ReAct/Plan-and-Execute/Reflexion/ToT。
Prompt模板结构。

### 3. 工具系统
工具定义格式(JSON Schema)、工具注册机制、工具选择策略。

### 4. 记忆系统
短期记忆/工作记忆/长期记忆，上下文管理策略（滑动窗口/摘要压缩）。

### 5. 多Agent协作
协作模式（顺序/并行/层级）、角色分工、通信机制。

### 6. 执行循环
ReAct循环控制、最大步数、错误恢复机制。

### 7. 可借鉴
前3个Agent架构设计亮点。
